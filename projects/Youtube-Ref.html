Youtube-Ref

Description

Youtube-Ref is a web app that is designed to store and playback specific points on Youtube videos. The user can create a reference by entering the link to the Youtube video and then using a slider to select the start and optional end point. After the video reference has been created the user can choose to delete or edit the reference. There is also a playlist feature to save videos into a playist and move videos between playlist. The inspiration for this app came from non implemented feature of my old capstone project which allowed the user to mark and save reference point on videos. (I am aware that there is a chrome extension that solves the same problem called TubeMark, but as this is a MERN stack app, it also allows syncing across different devices)

Challenges

The most difficult issue I faced during the development was implementing the backend and database for the playlist feature. I eventually settled for having the playlist and videos as different collections/tables in the database with a reference to a list of all the videos in the playlist collection and a reference to the playlistId in the video collection. Videos that were created without a playlist would have a playlistId of null which meant that they couldn't be ordered in the database.
